# Print
print的四種方式
## 1. print(x)
+   基本用法
## 2. print("x1=",x1,"x2=",x2)
+   我常用的方式
## 3. print("x1="+x1+"x2="+X2)
+   串接的內容物必須全部相同型別 ，如是字串要加上轉型。
## 4. print("x1=%d x2=%s" % (x1,x2))
+   類似C++的作法  %d=整數型別 %s=string %f=float
+   可以用於float取小數
+   |參數|用意|
    |---|:---:|
    |%d|整數|
    |%f|浮點數|
    |%1f|倍精度浮點數|
    |%c|字元|
    |%s|字串|

```python
    print("a=%.1f" % a) #取小數點後一位
```
