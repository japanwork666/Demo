# HTML 超文字標記語言
HyperText Markup Language
語法標籤以大小於符號成對。
例如:\<h1>這樣\</h1>
<h1>這樣</h1>

## 標題
h1~h6，數字越大，字型越小。

## 超連結
\<a href="網址">顯示名稱\</a>
<a href="https://www.google.com/">google</a>

## 段落
\<p>
段落中間可以塞很多東西
\</p>

<p>
第一段
</p>

<p>
第二段
</p>

## 圖片
可以給予寬高限制大小
<img src="./IMG/ps.png" width="200" height="100">

如果不給參數按照原圖大小

<img src="./IMG/ps.png">

## 標籤關鍵字
標籤內可以加入相關的關鍵字以達到想要的效果。
### 背景顏色
bgcolor="#03e8fc"

## JavaScript

\<script type="text/javascript">
    內容
\</script>
